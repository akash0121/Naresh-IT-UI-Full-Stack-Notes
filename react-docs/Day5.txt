Setup Bootstrap for project:

1. Install following libraries 
		
				> npm  install   bootstrap    bootstrap-icons --save

2. Link the following files to use bootstrap and its components

				/bootstrap-icons/font/bootstrap-icons.css
				/bootstrap/dist/css/bootstrap.css
				/bootstrap/dist/js/bootstrap.bundle.js

		Create a new React application using Build Tools [Bundling Tools]

- Build tools can create any application according to developer requirements.
- They can add resources and libraries required for application.
- They can setup environment for
		a) Build
		b) Debug
		c) Test &
		d) Deploy 
- They can setup local and cloud servers. 
- They can map to resources.
- They can configure your application for various environments
		a) Development
		b) Production 
- You can use various build tools like
		a) Webpack
		b) Vite
		c) Parcel 
		d) rs-build etc.

Create React app with Vite Bundler:

1. Open Terminal or command prompt 

2. Change to the location where you want to create a project.

3. Run the following command

					> npm  create    vite@latest   app_name -- -- template  react
