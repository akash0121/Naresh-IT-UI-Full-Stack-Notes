
                                                      Bootstrap 

- It is a powerful, extensible, rich featured front toolkit used to build interactive and responsive UI.

- It provides components and grid system to create responsive designs. 

- You can rapidly build interactive UI.


Setup Bootstrap for project:



1. Install following libraries



           > npm   install bootstrap --save

           > npm   install bootstrap-icons --save



    Note: The latest version of bootstrap is “5”, it doesn’t require jQuery explicitly. 

             Bootstrap versions upto 5 require jQuery. 

           

           > npm install jQuery --save         // not required for latest version



2. Link the following files to webpage



 <link rel=“stylesheet” href=“../node_modules/bootstrap-icons/font/bootstrap-icons.css”>

 <link rel=“stylesheet” href=“../node_modules/bootstrap/dist/css/bootstrap.css”>

 <script src=“../node_modules/bootstrap/dist/js/bootstrap.bundle.js”> </script>



Bootstrap Classes 



1. Box Model



           Margins

           .m-{size}            margin all directions size from 1 to 5. 

           .ms-{ }                left margin [s=start]

           .me-{ }                right margin [e-end]

           .mx-{ }                left and right 

           .mt-{ }                margin-top

           .mb-{ }                margin-bottom 

           .my-{ }                margin top and bottom 



           Padding

           .p-{size}

           .ps-{ }

           .pe-{ }

           .px-{ }

           .pt-{ }

           .pb-{ }

           .py-{ }



           Border

            .border                     base for border

            .border-{size}    

            .border-{contextual}        primary, secondary, danger, success, warning, info, dark ..



           Border Radius    

            .rounded 

            .rounded-{size}

            .rounded-circle

            .rounded-pill



           Width & Height

            .w-{size}            25, 50, 75, 100

            .h-{size}
